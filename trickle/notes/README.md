# DR Luxury Perfume Store

Website e-commerce parfum premium dengan desain mewah dan elegan.

## Fitur Utama

- **Katalog Produk Premium**: 7 parfum luxury dari brand ternama (Dior, Chanel, YSL, dll)
- **Shopping Cart**: Keranjang belanja dengan manajemen item lengkap
- **WhatsApp Integration**: Checkout langsung via WhatsApp
- **Animasi GSAP**: Transisi halus dan parallax effects
- **Responsif**: Optimal di mobile, tablet, dan desktop

## Teknologi

- React 18
- Tailwind CSS
- GSAP + ScrollTrigger
- Lucide Icons

## Struktur File

- `index.html` - Main entry point
- `app.js` - Root component dengan state management
- `data/products.js` - Product catalog
- `components/` - UI components (Navbar, Hero, Products, Cart, dll)
